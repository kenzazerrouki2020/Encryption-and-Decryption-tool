# Caesar Cipher Encryption & Decryption Tool

A simple Python program that uses the Caesar Cipher algorithm to encrypt and decrypt messages by shifting letters through the alphabet.

## Features

* Encrypt text messages
* Decrypt encrypted messages
* Supports uppercase and lowercase letters
* Keeps spaces, numbers, and symbols unchanged
* User-friendly input prompts

## How It Works

The program asks the user to:

1. Enter a message
2. Enter a shift number
3. Choose whether to encrypt or decrypt

Then it outputs the transformed message.

## Technologies Used

* Python
* `string` module

## Example

Input:
Message: Hello
Shift: 3
Choice: Encrypt

Output:
Khoor

This project is great for beginners learning:

* Python functions
